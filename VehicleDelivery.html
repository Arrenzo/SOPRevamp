Vehicle Delivery

This Standard Operating Procedure (SOP) describes the end-to-end process of import/inbound vehicle (RORO) cargoes discharged from the vessel, stored at yard, and delivered by truck.

Scenario Number: IMS521121411, IBS621121411

Steps

# Action Description Module User Manual Variation Optional
1 Create vessel A vessel serves as the profile of the physical vessel that contains information about its dimensions, storage locations, and maritime identifications. Visit Schedule > Vessel Create a vessel
2 Create vessel visit A vessel visit specifies the actual designated route of a vessel and its actual visit schedule to the terminal. Visit Schedule > Vessel Visit Create a vessel visit Create vessel service
Update visit operation mode
3 Submit manifest Submit a manifest to declare the list of onboard cargoes from an arriving vessel to its corresponding vessel visit.
Manifest Template: Import - All Types (XLSX) or General Cargo - Import (XLSX) 		Manifest and Documents > Manifest Submit a manifest Submit manifest (vessel visit)
4 Import discharge order from manifest A discharge order is a planned list of cargoes that are to be discharged from a vessel. Visit Schedule > Vessel Visit Import discharge order Create discharge order
5 Assign vessel location of discharge order The assigned location appears in Cargo Mobile to assist in tracking the vessel location of cargo during discharging operations.
Vessel locations are specified in the vessel profile. 		Visit Schedule > Vessel Visit Assign vessel location
6 Assign yard location of discharge order The assigned yard location appears in the Cargo Mobile app to assist in tracking the yard location of a cargo during discharging operations. Visit Schedule > Vessel Visit Assign yard location
7 Update vessel visit state to OK to Discharge Once the vessel arrives to the terminal and is ready for discharging operations, change the visit state to OK to Discharge. Visit Schedule > Vessel Visit Update the visit state of a vessel visit
8 Record vessel discharge - RORO [Mobile] Record the discharge operation of a vessel in its corresponding vessel visit. Cargo Mobile > Vessel Record vessel discharge Record offline vessel discharge - RORO [Mobile] Record vessel discharge - RORO (damaged) [Mobile]
9 Record vessel/crane activities [Mobile] Capture and manage real-time vessel activities that affect vessel operations and billing services. Cargo Mobile > Vessel Manage vessel activities Record vessel/crane activities Record vessel operation timestamps
10 Record vessel billable service Create and manage additional services performed for a vessel visit. Visit Schedule > Vessel Visit Manage vessel operation data
11 Update vessel visit state to Operation Completed or Departed Once all the operations for the vessel visit are completed, change the visit state to Operation Completed or once the vessel departs the terminal, change the visit state to Departed. Visit Schedule > Vessel Visit Update the visit state of a vessel visit
12 Confirm operation in vessel visit Once the operation data is retrieved for the vessel visit, confirm the operation data to finalize it. Visit Schedule > Vessel Visit Confirm operation data
13 Generate TDR A terminal departure report (TDR) is a report containing the summary of operations, productivity, and billable services provided to the vessel. Visit Schedule > Vessel Visit Generate terminal departure report
14 Settle seaside billing Generate invoice of seaside billing transactions for payment settlement. Billing > EBS System Order
15 View cargo inventory View cargo lots currently in the inventory according to their location in the terminal yard. GC/CFS > Cargo Inventory Cargo inventory monitoring
16 View cargo lot View and manage cargo lots, such as applying and removing of holds. GC/CFS > Cargo Lot Cargo lot management Apply cargo lot hold
Release cargo lot hold
17 View cargo tracking View each cargo lots and their movement in the terminal. GC/CFS > Cargo Tracking Cargo tracking
18 Update inventory - RORO [Mobile] Update the cargo details of RORO cargo currently in the inventory. Cargo Mobile > Yard Update RORO cargoes in the inventory
19 Update location - RORO [Mobile] Update the storage location of RORO cargo currently in the yard. Cargo Mobile > Yard Update RORO location
20 Admeasure - RORO [Mobile] Record the measurement of a RORO cargo in order to document the actual cargo admeasurement volume. Cargo Mobile > Yard Record RORO cargo admeasurement
21 Record cargo additional service - RORO [Mobile] Cargo Mobile > Yard
22 Inspect - RORO [Mobile] Record vehicle inspection in order to document the checking of damages, discrepancy, vehicle details, and the availability of interior and exterior parts, and accessories. Cargo Mobile > Yard Record a vehicle inspection
23 Release B/L Update the customs release status of the B/L from Not Released to Full Released.
Applies only to B/L records that belong to international vessel visits. 		Manifest and Documents > B/L Release a bill of lading Assign B/L - RORO
24 Create cargo delivery request Create a cargo delivery request to manage international or domestic delivery transactions. GC/CFS > Request Create a cargo delivery request
25 Dispatch service request Dispatch a service request when it is ready to start with the required terminal operation.
Applies only to service requests with postpaid invoice option. 		GC/CFS > Request Dispatch a service request
26 Settle landside billing Generate invoice of landside billing transactions for payment settlement. Billing > EBS OnSpot Clearance System Order
27 Create truck visit Applies only if terminal supports truck visit mode. GC/CFS > Truck Visit Create gate pass Print truck visit slip
28 Confirm truck arrival [Mobile] Confirm and record the arrival date and time of incoming truck or driver. Cargo Mobile > Gate Confirm a truck arrival
29 Record truck loading - RORO [Mobile] Record the loaded quantity of the cargo from a location in the yard inventory to the truck. Cargo Mobile > Gate Record a truck loading
30 Complete mission [Mobile] Complete the mission to finish the loading or unloading operations for a truck mission. Cargo Mobile > Gate Complete a truck mission Print CIR
31 Confirm truck departure [Mobile] Confirm and record the departure date and time of outgoing truck or driver. Cargo Mobile > Gate Confirm a truck departure
